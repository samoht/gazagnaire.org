dyntype / ORM / Shelf
A collection of tools using dynamic types in OCaml. Dyntype is a Camlp4 extension adding dynamic types to OCaml. ORM uses dynamic types to provide a complete and efficient Object-Relationship Mapping to SQlite3 for OCaml. Shelf uses dynamic types to provide a collection of parsers and printers to JSON, S-expressions, HTML, ...
