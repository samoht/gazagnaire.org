Xen API
Xen API (or xapi) is a management stack that configures and controls Xen-enabled hosts and resource pools, and co-ordinates resources within the pool. Xapi exposes the Xen API interface for many languages and is a component of Citrix XenServer. Xen API is written mostly in OCaml.
[xen-org.git][website]
